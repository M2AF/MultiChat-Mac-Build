HOW TO FIND YOUR CHANNEL ID
1. Go to abstools.top and log in
2. Click Chat Overlay in the left sidebar
3. Press F12 to open browser DevTools
4. Click the Network tab → filter by WS
5. Refresh the page — you'll see a connection starting with ws2channel_id=
6. Click it → copy the full URL from the Headers tab
7. Paste it in the box above
It will look like this:
wss://abstools.top/api/ws?channel_id=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
